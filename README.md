# My Turing Friend

## A 2D platformer game written with the Godot Engine.

**My Turing Friend** is a 2D platformer built from scratch in just over 7 days for the **Godot Wild Jam #53**, where the theme was *"Assembly Required"*. 

Driven by curiosity and a desire to build a complete game on my own, this project was a fantastic learning experience in time management, scope control, and getting hands-on with the Godot Engine.

## 🎮 Development Journey
* **The Art Challenge:** Initially hoping to team up with a 2D artist, I ended up creating all the graphics myself despite having no prior digital art experience. Over 80% of the jam time was spent crafting the visuals, with the main hero's art being finalized on the very last day!
* **Four-Color Limitation:** To make things even more interesting, I adopted the jam's "Four Color" limitation card, which dictated the entire retro aesthetic of the game.
* **Scope & Mechanics:** The original vision was highly ambitious, including mechanics for resource gathering, shield charging, and temperature control. Ultimately, the scope was scaled back to ensure a polished, playable experience within the tight deadline—a great lesson in game jam survival.
* **The Engine:** Godot proved to be incredibly user-friendly, and its built-in 2D platformer tutorials were instrumental in getting the core mechanics off the ground quickly.
